== Doxygen

https://www.doxygen.nl/

https://www.doxygen.nl/manual/faq.html

----
❯ cd dev/doxygen

❯ doxygen -g       


Configuration file 'Doxyfile' created.

Now edit the configuration file and enter

  doxygen

to generate the documentation for your project


----

----
❯ rm -rf docs/html docs/latex
----

----
❯ doxygen -x                 
# Difference with default Doxyfile 1.9.8
PROJECT_NAME           = Maelstrom
OUTPUT_DIRECTORY       = /workspaces/libsdl-org-Maelstrom/dev/doxygen/docs
EXTRACT_ALL            = YES
INPUT                  = /workspaces/libsdl-org-Maelstrom/ \
                         /workspaces/libsdl-org-Maelstrom/maclib \
                         /workspaces/libsdl-org-Maelstrom/netlogic \
                         /workspaces/libsdl-org-Maelstrom/screenlib
UML_LOOK               = YES
CALL_GRAPH             = YES
CALLER_GRAPH           = YES
DOT_IMAGE_FORMAT       = svg
INTERACTIVE_SVG        = YES
----
